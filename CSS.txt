На странице стили подключаются:
1) <p style="..."></p>
2) <head>
	<style>
	...
	</style>
   </head>
3) В <head>-теге <link href="url" rel = "stylesheet">

Описание стиля внутри тега элемента: <p style="background-color:red; font-size:10px;">Some text</p>
Во внешних файлах или теге <style>:

для всех модулей определенного типа:
p {свойство: значение;	свойство: значение;	свойство: значение;	...}

для поименованных модулей ( class="specialselectorname")
.specialselectorname {свойство: значение;	свойство: значение;	свойство: значение;	...}


если в имени селектора вместо точки стоит # (#selectorName)- это будет стиль УНИКАЛЬНОГО элемента.
Это позволит JavaScript находить данный элемент.

Преимущетсво всегда имеет inline стиль, затем описанный ниже по коду, ближе к блоку:
глобальный, или из отдельного файла css, смотря кто в header ниже написан.



Примеры записи стилей:
font: italic bold 14px Arial;

background-image: url(./images/img1.png), url(./images/img2.png);	картинка 1 наложится на картинку 2

background-repeat: no-repeat;			не повторяться фоновой картинке, 
repeat-x, repeat-y, 
space, 					равномерно размещает картинки с пробелами между ними
round					!! размещает целое число картинок (без разрезов, но с растягиванием)
background-size: contain			масштабирует картинку сохраняя пропорции так, чтобы она полностью поместилась в элемент, у элемента может остаться пустое пространство
cover						установит картинку так, чтобы она полностью заняла элемент, но отчатси не поместится
background-attachment: scroll			скроллится ли фон?  scroll - скроллится, fixed - фиксированный, local - в зависимости от поведения элемента
background-origin: padding-box			ставится к краю элемента с учётом толщины границы
border-box				относительно границы
content-box				относительно контента

border-image: url(./images/img1.png) 40 round round;		углы картинки ставим в качестве границ элемента, вырезая их как квадраты со стороной 40 pxl
transparent						атрибут для прозрачной рамки
border-radius: 70px 50px 70px 40px/20px;			скругление 4 углов элемента рамкой
outline: 2px solid blue;				внешняя граница элемента, не влияет на размер элемента, ставится для всех сторон
border-right: 3px dotted darkorange;		рамка правая, 3 пикселя, пунктирная, оранжевая

npm create-react-app		-создать папку-проиложение на React (как вариант, начать с npx)
npm start 			-запустить сервер разработки
Ctrl + C			- отключить сервер
npm run build			- собирает приложение в финальное состояние для продажи
npm test 			- запускает тесты

npm install react-router-dom --save-dev		- установить роутеры с сохранением в файле dependencies
npm install react-router-dom			- установим роутеры
npm install react-cookie			- установим куки


Deployment:
npm install -g expo-cli			- установка приложения Expo для тестирования нашего приложения в Android и IOS
cd <желаемая папка для нашего приложения>
expo init <имя нашего front-end приложения>	- создание фронт-енд приложения в expo
Expo скачать на телефоны и таблеты и подключиться к нашему expo-приложению.

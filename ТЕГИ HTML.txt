Блочные элементы (занимают всю ширину страницы).
<div></div> - блок, который может содержать что угодно
<p></p> - блок абзаца, отображается так, как требуют атрибуты и правила HTML, например, только с 1 пробелом между словами.
атрибуты: dir="rtl" - выравнивание справа
<pre></pre> - блок отформатирвоанного текста, показыает текст со всеми переносами строк, пробелами, заданными в коде, моноширинным шрифтом.
<blockquote></blockquote> - абзац цитаты, 8-пиксельный отступ слева
<h1></h1> - абзац заголовка 1 уровня.  Всего уровней 6. h1  - 1 на странице,  h2 - 2-4 штуки, h3 - 4-8 штук.

<hr/> - горизонтальная линия. Horizontal rule
<br/> - перенос строки. Break



Теги физической разметки:
<b></b> - жирное начертание текста
<i></i> - курсив
<s></s> - перечеркнутый
<bdo dir="rtl"> <bdo/> - разворачивает содержимое задом наперёд (Дом - моД)
<small></small> - размер на 1 единицу меньше (размер от 1 до 7, по умолчанию: 3)
<big></big>  -  размер на 1 единицу больше
<sup></sup> - верхний индекс (можно запомнить по кружку сверху)
<sub></sub> - нижний индекс (можно запомнить по кружку снизу)
<meter></meter> - вывод значения из известного диапазона
<progress></progress> - отображение прогресса завершённости
<span></span> - выделяет текст для стилевого оформления с помощью CSS


Теги логической разметки:
<abbr></abbr> - указывает, что текст - аббревиатура. Атрибут title указывет текст всплывающей подсказки.
<cite></cite> - сноска или цитата
<code></code> - текст, содержащий програмный код
<strong></strong> - выделяет текст жирным и выделит его для программ (например, читалок текста)
<em></em>   - опять же логически выделяет кусок текста, в браузерах выделяется курсивом.
<del></del> - обозначает удалённые элементы в версиях документа. В браузерах перечёркивает текст.
<ins></ins> - добавленный текст, в браузерах подчёркивается.
<mark></mark> - выделяет фрагмент текста. В браузерах - жёлтым фоном.
<var></var> - объявление переменных


<a></a> - создает ссылку на документ, страницу сайта, якорь или внешний ресурс
href=  указывает адрес перехода
title="GOOGLE"    	всплывающая подсказка 
accesskey="t" 		комбинация горячих клавиш, по которым сработает ссылка
target="_blank" 	указывает, в каком окне нужно открыть ссылку blank или self

Создание нескольких одинаковых элементов в IDE позволяет реализовать знак умножения, звёздочка: *
p*4 - создать 4 новых параграфа.


<img/> тег для размещения изображения
src=""   	атрибут для указанеия источника картинки
alt=""		значение выводится вместо картинки, если та не загрузится
width и height - высота и ширина.  Надо соблюдать пропорции.
<map></map> - карта с областями, которые описываются тегом <area/>


<table></table> 		таблица
<thead></thead>			верхняя часть таблицы
<tbody></tbody>			центральная цасть таблицы
<tfoot></tfoot>			нижняя часть таблицы
<tr></tr> 			строка
<td></td>			ячейка
<th></th>			ячейка с жирным начертанием текста
арибут colspan="3"		объединение колонок в этой ячейке (ячеек по горизонтали)
арибут rowspan="2"		объединение строчек в этой ячейке (ячеек по вертикали)
арибут align="center"		выравнивание текста по центру

<colgroup></colgroup>		группирует один или более столбцов для единого форматирования
<col></col>			формирует группы столбцов, которые делят таблицу на разделы


<ul></ul>			маркированный список
атрибут type="circle"		вид маркера. disc, circle или square
<ol></ol>			нумерованный (упорядоченный) список
атрибут type="A"		вид маркера. A, a, I или i
start=				начало нумерации
reversed=
value=				задаёт конкретное значение нумерации
<li></li>			один элемент списка
